# MVP Scope & Task Tracker - Website OCTAGRAM

Dokumen ini berisi daftar fitur minimal (MVP) yang harus diselesaikan untuk peluncuran versi pertama website OCTAGRAM. 

## 📌 1. Header & Navigasi
Bagian paling atas yang selalu terlihat di setiap halaman.
- [ ] Pasang Logo OCTAGRAM (Gunakan format SVG, warna `#0098ff`).
- [ ] Buat Menu Navigasi statis: `Layanan`, `Karya`, `Tentang`, `Kontak`.
- [ ] Buat tombol CTA di pojok kanan: "Konsultasi" (Background `#0098ff`, Text `#ffffff`).
- [ ] Pastikan *hamburger menu* berfungsi dengan baik di tampilan *mobile*.

## 📌 2. Hero Section
Kesan pertama pengunjung saat web baru dibuka.
- [ ] Tambahkan Headline utama: "Solusi Digital Terjangkau Untuk Bisnis & Kreativitas Anda."
- [ ] Tambahkan Sub-headline: "Jasa pembuatan Website, Mobile App, dan UI/UX untuk UMKM dan Pelajar. Harga bersahabat, kualitas profesional."
- [ ] Pasang aset visual/ilustrasi di sebelah kanan (atau di bawah teks untuk versi mobile).
- [ ] Buat 2 tombol CTA: 
  - Tombol Utama: "Mulai Proyek" (Warna `#0098ff`).
  - Tombol Sekunder: "Lihat Layanan" (Garis tepi / *outline* `#0098ff`, latar `#ffffff`).

## 📌 3. Layanan / Services Section
Penjelasan 4 jasa utama OCTAGRAM beserta *range* harga.
- [ ] Buat *Grid Layout* untuk 4 Kartu Layanan:
  - [ ] **Card 1: Web Development** (Icon Web, "Mulai Rp 250k").
  - [ ] **Card 2: Mobile Development** (Icon HP, "Kotlin & Flutter", "Mulai Rp 1.000k").
  - [ ] **Card 3: Desktop App** (Icon PC, "Mulai Rp 700k").
  - [ ] **Card 4: UI/UX Design** (Icon Figma, "Mulai Rp 150k").
- [ ] Pastikan warna *background section* ini menggunakan `#f0faff` (Biru Muda) sebagai variasi agar tidak menyatu dengan Hero.

## 📌 4. Portofolio & Workflow
Menampilkan kredibilitas dan cara kerja tim.
- [ ] Siapkan 3-4 *thumbnail* gambar proyek yang pernah dibuat (Desain Figma, Mockup Web, dll).
- [ ] Buat *Grid/Carousel* sederhana untuk menampilkan karya.
- [ ] Buat *section* Workflow singkat dengan 4 ikon langkah: 
  1. Diskusi
  2. Desain (Figma)
  3. Koding 
  4. Rilis
  
## 📌 5. Tentang Kami & Kontak
Informasi tim dan *call-to-action* penutup.
- [ ] Tulis deskripsi singkat tentang tim OCTAGRAM.
- [ ] Buat Form Kontak sederhana: Nama, Email, Kebutuhan Jasa, Pesan.
- [ ] Buat tombol CTA melayang (Floating Button) khusus WhatsApp di pojok kanan bawah.
- [ ] **Footer:** Tambahkan *copyright* "© 2026 OCTAGRAM Freelance" dan *link* sosial media.
